# 📋 Survey Form

A clean, responsive survey form built with vanilla HTML, CSS, and JavaScript.

## 🌐 Live Demo
[View Live Site](https://cmatver-2.github.io/survey-form)

## ✨ Features
- Name, Email, Phone, and Message fields
- Strict input validation (email format, 10-digit phone)
- Duplicate email and phone detection
- Message limited to 200 words
- Entries saved in browser localStorage
- Submissions list with message preview (100 chars)
- Toast notifications (top-left)
- Confirmation message on successful submit

## 🗂️ Folder Structure

```
survey-form/

```

## 🛠️ Built With
- HTML5
- CSS3
- Vanilla JavaScript

## 🚀 How to Run Locally
1. Clone the repo
```bash
   git clone https://github.com/cmatver-2/survey-form.git
```
2. Open `index.html` in your browser
